# **mitlab**
---

##**Prerequisite**
**install nodeJS**<br>
http://nodejs.org/<br><br>
**install bower**<br>
`npm install -g bower`
##**How to build it**
**Step 1.<br>**
`git clone https://github.com/s850042002/mitlab.git`<br><br>
**Step 2.<br>**Inside the directory and <br>
`npm install`<br><br>
**Step 3.<br>** go to the `public` directory and <br>
`bower install`<br><br>
**Step 4.<br>** Start the web<br>
`node bin/www`

####It works.<br>
http://localhost:3000
