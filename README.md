mitlab
==

##**Prerequisite**
**install nodeJS**
http://nodejs.org/

**install bower**
`npm install -g bower`

##**How to build it**
**Step 1.**

`git clone https://github.com/s850042002/mitlab.git`

**Step 2.**

Inside the directory and

`npm install`

**Step 3.**

go to the `public` directory and

`bower install`


**Step 4.**

 Start the web

`node bin/www`

####It works.
http://localhost
